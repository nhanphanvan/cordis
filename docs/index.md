# Cordis Documentation

Cordis is a file and artifact management platform with a FastAPI backend and a Typer-based CLI. This documentation is the primary entrypoint for developers working on the codebase or integrating with the service.

## Start Here

- [Getting Started](./getting-started.md): install dependencies, run the backend, inspect the CLI, and complete a first local workflow
- [Configuration](./configuration.md): backend environment settings, CLI config files, workspace registration, and cache behavior
- [CLI Guide](./cli.md): command groups and common operator workflows
- [Backend API](./backend-api.md): guide-first walkthrough of the current HTTP API surface
- [Transfer Workflows](./transfer-workflows.md): detailed upload and download behavior across the CLI, backend, storage, and cache

## Engineering Guides

- [Architecture](./architecture.md): backend and CLI structure, request flow, and storage boundary
- [Data Model](./data-model.md): main entities and how they relate
- [Development](./development.md): code organization, contribution workflow, and where new code should live
- [Testing](./testing.md): test layout, verification commands, and targeted test execution
